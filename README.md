# QR Code Generator

A simple QR Code Generator built with HTML, CSS and JavaScript.

## Features
- Generate QR code from any URL
- Input validation for empty and invalid URLs
- Clean dark theme UI

## Technologies Used
- HTML
- CSS
- JavaScript
- QRCode.js Library

## How to Use
1. Enter a valid URL (with http:// or https://)
2. Click "Submit" button
3. Scan the generated QR code!
